# My VendingMachine Java Project
This project is a simple Java application that demonstrates the use of a software vending machine.

## Prerequisites

JDK

* IntelliJ IDEA 2023.1
* Java Development Kit (JDK) 1.8 or higher
* Javafx 20.0.1
* apache-maven-4.0.0
* MACos
